## Robust Subspace Learning Based Orthogonal Non-negative Tucker Decomposition for Incomplete Multi-view Clustering (SONTDML)
The first paper titled “Robust Subspace Learning Based Orthogonal Non-negative Tucker Decomposition for Incomplete Multi-view Clustering (SONTDML)”. The [***SONTDML file***](./SONTDML) shows the sample code of "Title: Robust Subspace Learning Based on Non-negative Tucker Decomposition for Incomplete Multi-view Clustering".

**Datasets:**

+ **YaleA** - *default parameter:* $\lambda_{1} = 3,\lambda_{2} = 0.01,\lambda_{3} = 0.001, \lambda_{4} = 10, \beta = 0.1$
+ **MSRC v1** - *default parameter:* $\lambda_{1} = 0.001,\lambda_{2} = 0.1,\lambda_{3} = 0.1, \lambda_{4} = 0.1, \beta = 0.1$
+ **MSRC v6** - *default parameter:* $\lambda_{1} = 0.001,\lambda_{2} = 0.1,\lambda_{3} = 0.005, \lambda_{4} = 0.01, \beta = 0.1$
+ **COIL20** - *default parameter:* $\lambda_{1} = 0.01,\lambda_{2} = 0.1,\lambda_{3} = 1, \lambda_{4} = 0.1, \beta = 0.1$
+ **100Leaves** - *default parameter:* $\lambda_{1} = 0.01,\lambda_{2} = 0.1,\lambda_{3} = 0.01, \lambda_{4} = 0.01, \beta = 0.01$
+ **Hdigit** - *default parameter:* $\lambda_{1} = 0.0001,\lambda_{2} = 0.0001,\lambda_{3} = 0.0001, \lambda_{4} = 10, \beta =10$
